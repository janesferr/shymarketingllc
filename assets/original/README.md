# Original Site Asset Mirror

These files were mirrored from the public assets used on `https://www.shymarketingllc.com/` so the rebuild can reuse the original media.

## Included

- Logos and favicon
- Hero images
- Service/resource card images
- Partner images
- Testimonial portraits
- WhatsApp graphic
- Video poster and hosted MP4

## Notes

- Files in this folder are intended as the original-source asset set.
- The current rebuild still uses lightweight placeholder SVG branding in a few places; these mirrored files are now available to swap in wherever needed.
- Third-party app bundles, fonts, and builder scripts were not vendored into the repo because they are not needed for a Lighthouse-focused rebuild.
